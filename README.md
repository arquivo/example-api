# Arquivo.pt API Example

A minimal JavaScript example showing how to query the [Arquivo.pt](https://arquivo.pt) web archive API and display results in a webpage.

## What it does

Opens a search against the Arquivo.pt full-text search API and renders the results (title, year, and original URL) using Bootstrap cards.

## Usage

Just open `example.html` in a browser — no build step or server required.

## API Documentation

See the full [Arquivo.pt API docs](https://arquivo.pt/api) for available endpoints and parameters.
